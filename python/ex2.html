Exercices if-else¶

Les exercices ci-dessous sont écrits avec des boucles et des expressions conditionnels. Essayez de trouver la solution sans cliquer sur le bouton click to show (+).

Comparaison¶

Quel est le code pour afficher ces 8 lignes de comparaisons?
(2 lignes de code suffisent)

for i in range(8):
    print(i, '> 3 =', i > 3)

0 > 3 = False
1 > 3 = False
2 > 3 = False
3 > 3 = False
4 > 3 = True
5 > 3 = True
6 > 3 = True
7 > 3 = True

Plus grand¶

Quel est le code pour afficher ces comparaisons ?

for i in range(8):
    if i > 3:
        print(i, 'plus grand que 3')
    else:
        print(i)

1
2
3
4 plus grand que 3
5 plus grand que 3
6 plus grand que 3

Pair/impair¶

Utilisez l’opérateur modulo % pour afficher si un nombre est pair ou impaire ?

for i in range(8):
    if i % 2 == 0:
        print(i, 'est pair')
    else:
        print(i, 'est impair')

0 est pair
1 est impair
2 est pair
3 est impair
4 est pair
5 est impair
6 est pair
7 est impair
