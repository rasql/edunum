Graphes¶

Le module matplotlib permet d’afficher des graphes. Le module numpy permet de faire du calcul numérique avec des matrices.

Pour utiliser ces modules nous devons les importer. Il est tradition de les importer sous les noms raccourcis suivants.

  • plt pour pyplot

  • np pour numpy

import matplotlib.pyplot as plt
import numpy as np

Créons une liste de données que nous pouvons afficher.

y = [2, 4, 3, 6, 3]

Un graphique simple (plot en anglais) est obtenu avec ceci.

plt.plot(y)

[<matplotlib.lines.Line2D at 0x7fd4222a65d0>]

L’objet retourné est de type Line2D. Dorénavant nous terminons la dernière ligne avec un point-virgule ; pour supprimer cette information.

Attention:
Pour afficher un graphique dans votre programme Python, vous devez ajouter la commande plt.show() pour l’afficher dans une fenêtre à part. Dans ce document-ci cette commande est inclue de façon implicite.

Titres et légendes¶

Nous pouvons ajouter un titre ainsi que des légendes pour les deux axes x et y.

plt.title('Evolution')
plt.xlabel('temps')
plt.ylabel('montant')
plt.plot(y);

Style des lignes¶

Nous pouvons également ajouter un string de formatage. Les couleurs sont indiqué par

  • r rouge

  • g vert (green)

  • b bleu

  • y jaune (yellow)

  • m magenta

  • c cyan

  • k noir (black)

  • w blanc (white)

plt.plot(y, 'mo-');

La forme des points est indiqué par

  • o cercle

  • s carré (square)

  • ^ triangle (haut)

  • v triangle (bas)

  • * asterisque

  • x x

  • + plus

plt.plot(y, 'gv:');

Le format de ligne est indiqué par

  • - solide

  • : pointillé

  • -- traitille

  • -. mixte

plt.plot(y, 'cs-.');

Nous pouvons mélanger deux styles. Pour ceci nous devons afficher deux lignes.

plt.plot(y, 'b:', y, 'or')

[<matplotlib.lines.Line2D at 0x7fd423de44d0>,
 <matplotlib.lines.Line2D at 0x7fd423de4550>]

Coordonnées dans x¶

Si nous affichons un graphe à partir d’un seul vecteur, les valeurs x sont des entiers commençant à 0. Nous pouvons définir notre propre plage x, par exemple avec un vecteur x allant de -2 à +2.

x = [-2, -1, 0, 1, 2]
plt.plot(x, y,'k^:');

Propriétés de ligne¶

Avec des arguments à options nous pouvons changer les propriétés d’une ligne.

plt.plot(x, y, color='r', 
         linewidth=2, linestyle='--', 
         marker='o', markersize=12, 
         markerfacecolor='yellow', 
         markeredgewidth=2, markeredgecolor='blue');

Histogramme¶

Un diagramme à barres est aussi appelé histogramme. Il représente des informations sous forme de barres.

plt.bar(x, y)

<BarContainer object of 5 artists>

Un diagramme à barres produits un objet BarContainer qui contient des objets Rectangle.

for x in _:
    print(x)

Rectangle(xy=(-2.4, 0), width=0.8, height=2, angle=0)
Rectangle(xy=(-1.4, 0), width=0.8, height=4, angle=0)
Rectangle(xy=(-0.4, 0), width=0.8, height=3, angle=0)
Rectangle(xy=(0.6, 0), width=0.8, height=6, angle=0)
Rectangle(xy=(1.6, 0), width=0.8, height=3, angle=0)

Multiples lignes¶

Nous pouvons placer multiples lignes sur un plot.

t = np.arange(0., 2., 0.2)
plt.plot(t, t, 'r--', t, t**2, 'bs:', t, t**3, 'g^:')

[<matplotlib.lines.Line2D at 0x7fd422210d10>,
 <matplotlib.lines.Line2D at 0x7fd4240c7050>,
 <matplotlib.lines.Line2D at 0x7fd4240c7290>]

Courbes trigonométriques¶

Le module numpy nous permet l’accès faciles aux fonctions trigonométriques.

t = np.linspace(0, 4*np.pi, 100)
plt.plot(t, np.sin(t), label='sin(t)')
plt.plot(t, np.cos(t), label='cos(t)')
plt.legend()
plt.grid()

Ici nous affichons le carré du sinus.

\[ f(t) = (\sin t)^2 \]
t = np.linspace(0, 4*np.pi, 100)
plt.plot(t, np.sin(t), label='sin(t)')
plt.plot(t, np.sin(t)**2, label='sin(t)**2')
plt.legend()
plt.grid()

Figures de Lissajous¶

Les figures des Lissajous sont des combinaisons de courbes sinus et cosinus.

t = np.linspace(0, 2*np.pi, 100)
plt.grid()
plt.plot(t, np.sin(3*t), 'r');

plt.grid()
plt.plot(np.sin(t), np.cos(t), 'r');

plt.grid()
plt.plot(np.sin(t), np.cos(3*t), 'r');

plt.grid()
plt.plot(np.sin(2*t), np.cos(3*t), 'r');

Echelle logarithmiques¶

Nous pouvons afficher une axe

y = np.arange(0, 100, 2)
plt.grid()
plt.plot(y, 'o-')
plt.yscale('log')

plt.grid()
plt.plot(y, 'o-')
plt.xscale('log')

plt.grid()
plt.plot(y, 'o')
plt.xscale('log')
plt.yscale('log')
